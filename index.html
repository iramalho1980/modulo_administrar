📋 Introdução ao Módulo Administrar

▼

O Módulo Administrar é uma ferramenta poderosa que permite gerenciar o tempo de inatividade dos usuários no sistema, oferecendo recursos avançados para controle e análise de produtividade.

Principais Funcionalidades:

⚙️ Configuração de Parâmetros

▼

A configuração dos parâmetros é o primeiro passo para utilizar o módulo de forma eficiente. Siga os passos abaixo:

Menu → Controle → Parâmetros
1 Controlar o Tempo no Sistema

Na janela de parâmetros, selecione o quadro "Controlar o tempo no sistema" para habilitar as demais opções disponíveis.

2 Definir Tempo de Ociosidade

Na opção "Não contar como tempo gasto quando o sistema ficar Ocioso por X minutos", defina quantos minutos após o usuário ficar ocioso será considerado nos relatórios.

Exemplo prático:
Se o usuário ficar 50 minutos ocioso no sistema e você definir 10 minutos nessa opção, será considerado somente os 10 minutos. Os outros 40 minutos não serão contabilizados como tempo gasto.
3 Configurar Exibição de Tela de Ociosidade

Na opção "Não exibir a tela de ociosidade":

  • Marcado: Não será demonstrado que o usuário está ocioso
  • Desmarcado: Será apresentada a mensagem quando o usuário ficar ocioso
4 Desconexão Automática

Na opção "Desconectar automaticamente usuário após X minutos de ociosidade", defina quantos minutos após o usuário ficar ocioso ele será desconectado do sistema automaticamente.

5 Selecionar Usuários Afetados

Após definir a quantidade de tempo, clique no botão "Usuários" e selecione os usuários que você deseja que essa regra afete.

6 Salvar Configurações

Clique no botão "OK" para salvar as configurações.

⚠️ Atenção: Esta alteração só terá efeito após reiniciar todos os módulos do sistema. Certifique-se de reiniciar o sistema após salvar as configurações.

📂 Classificações de Atividades

▼

As classificações permitem categorizar atividades realizadas fora do sistema para melhor organização e análise.

Menu → Arquivos → Classificações
1 Incluir Nova Classificação

Clique no botão "Novo" para criar uma nova classificação de atividade.

2 Código Automático

O código será automaticamente preenchido conforme as classificações já cadastradas dentro do sistema.

3 Descrição da Classificação

No campo "Descrição", informe uma descrição clara e objetiva para essa classificação.

4 Salvar

Clique no botão "Gravar" para salvar a nova classificação.

Dica: Utilize classificações que façam sentido para o seu fluxo de trabalho, como "Reunião Externa", "Atendimento Presencial", "Treinamento", etc.

📝 Registro de Atividades Externas

▼

O registro de atividades permite cadastrar todas as atividades que não foram realizadas dentro do sistema, mas que devem ser consideradas no tempo gasto nos relatórios do Módulo Administrar.

Menu → Arquivos → Registro de Atividades
1 Novo Cadastro

Clique no botão "Novo" para iniciar o registro de uma nova atividade.

2 Informar Cliente

No campo "Cliente", informe o código do cliente para qual realizou a atividade.

3 Selecionar Classificação

No campo "Classificação", selecione o tipo de atividade realizada para esse cliente.

4 Data e Horários

Preencha os seguintes campos:

  • Data: Data em que foi realizada a atividade
  • Hora Inicial: Horário em que foi iniciada a atividade
  • Hora Final: Horário em que foi encerrada a atividade
5 Status da Atividade

No campo "Finalizar", selecione se a atividade foi finalizada ou não.

6 Descrição

No campo "Descrição", informe uma descrição detalhada da atividade realizada.

7 Salvar Registro

Clique no botão "Gravar" para salvar o cadastro da atividade.

📊 Relatório de Tarefas Realizadas

▼

Este relatório permite avaliar uma relação de lançamentos contábeis e fiscais realizados por todos os usuários do sistema.

Menu → Relatórios → Tarefas Realizadas

Configurações do Relatório:

1 Modelo do Relatório

No quadro "Modelo do Relatório", defina se o relatório será Completo ou Resumido.

2 Período

No quadro "Data", informe o período inicial e final em que serão filtradas as informações.

3 Quebra de Relatório

No quadro "Quebra por", selecione:

  • Cliente + Usuário
  • Usuário + Cliente
4 Tipo de Relatório

No quadro "Tipo de Relatório", selecione:

  • Somente lançamentos contábeis
  • Somente notas fiscais
  • Todos
5 Ordenação

Defina se a ordem será pelo código ou nome do cliente.

6 Origem dos Lançamentos

No quadro "Origem Lançamentos Contábeis", defina se deseja filtrar uma origem específica ou selecione a opção "Todos".

7 Considerar Data

No quadro "Considerar", selecione se será considerada:

  • Data do lançamento
  • Data do documento
8 Selecionar Usuários

No quadro "Usuário", selecione de quais usuários você deseja avaliar as tarefas realizadas.

9 Filtrar Clientes

Clique no botão "Cliente" para selecionar um cliente específico ou selecione "Todos" para gerar de todos os clientes.

10 Emitir Relatório

Após configurar todos os filtros, clique no botão "OK" para emitir o relatório.

Resultado: O relatório será gerado conforme os filtros realizados, apresentando todas as informações de forma organizada e detalhada.

📈 Relatório de Registro de Atividades

▼

Este relatório permite avaliar as atividades realizadas fora do sistema que foram registradas manualmente.

Menu → Relatórios → Registro de Atividades
1 Definir Período

Defina um período inicial e final para filtrar no relatório.

2 Ordem do Relatório

No quadro "Ordem", defina a ordem em que serão demonstradas as informações no relatório.

3 Quebra de Relatório

No quadro "Quebrar Relatórios", defina se terá alguma quebra no relatório.

4 Filtros Específicos

Caso precise filtrar informações específicas, utilize os botões:

  • Clientes - Para filtrar por cliente específico
  • Usuários - Para filtrar por usuário específico
  • Classificações - Para filtrar por tipo de atividade
5 Gerar Relatório

Após configurar os filtros desejados, clique em "OK" para gerar o relatório.

Dica: Utilize a opção "Todos" quando desejar uma visão geral de todas as atividades registradas no período selecionado.
